Microsoft JDBC-ODBC Bridge   Introduction

 


Introduction

The Microsoft JDBC-ODBC bridge is part of the core set of classes that come with the Microsoft VM for Java.

Requirements

In order to use the JDBC/ODBC bridge you need to have the following installed:

  1. ODBC 3.0 or later.
  2. At least 1 ODBC driver for the type of data you need. For example, if you want to get to SQL Server data, you must have the appropriate ODBC SQL Server driver installed.

Running the JDBC sample

To run the JDBC Samples please follow the instructions in the Samples section.
