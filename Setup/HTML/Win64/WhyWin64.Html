Why should you care about 64-Bit Windows NT?

First let's look at what a 64-bit Windows NT system will do that is different from what happens today on a 32-bit Windows NT system.

A 64-bit system will support far more physical memory that a 32-bit system. More memory translates to greater scalability — for applications, for users, for data, and for performance. In simple numbers, a 32-bit Windows NT system can support 4 Gbytes of memory. A 64-bit Windows NT system will support 16 TB of memory.

  1. A 64-bit Windows NT system can provide more memory to run applications .

    The amount of available memory in a 64-bit Windows NT system is realistically constrained only by the physical capacity of the system to hold the memory boards. This abundance of physical memory translates into several application benefits.

    Each application on the system can support more users . All or part of each application on any system must be replicated for each user. This replication requires memory. On a 64-bit Windows NT system, many more users can access any given application because much more memory can be supported on the system.

    The applications can run at much higher levels of performance on a 64-bit Windows NT system. Again, the amount of physical memory supported on a 64-bit system allows many more applications to run simultaneously and remain completely resident in the system's main memory. This reduces or eliminates the performance penalty of swapping pieces of applications to and from disk. The performance advantage of 64-bit Windows NT is especially evident in the largest applications, such as those in ERP or financial and scientific modeling.

  2. A 64-bit Windows NT system can provide much more memory for data storage and manipulation.

    Databases can store more of their data in the physical memory of the system. In some cases, entire databases could operate as memory-resident. Access to data will be faster since a disk read is not necessary. This also drives up the performance of the application using the database.

    Applications can manipulate data more easily and more reliably. Huge data structures can be created in memory. Video composition for motion picture work requires 64-bit systems today. Modeling for scientific and financial applications benefits greatly from memory resident data structures not possible on 32-bit systems. Today, these and a growing number of memory dependent applications are run on expensive UNIX workstations and servers. 64-bit Windows NT will bring these applications to industry-standard platforms.
