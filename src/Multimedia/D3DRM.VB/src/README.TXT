1.  \Mssdk\Samples\Multimedia\D3DRM.VB\src directory
--------------------------------------------------------

The \Mssdk\Samples\Multimedia\D3DRM.VB\src directory contains Visual Basic samples
which use RMControl.Ocx. This file must be generated by building the project located in
