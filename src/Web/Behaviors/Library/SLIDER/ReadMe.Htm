slider Behavior

This document describes technologies available in Internet Explorer 5. While we encourage you to evaluate these features and to send us your feedback, please note that these features are subject to change.

The slider behavior inserts a slider control into a Web page and is a method for Web authors to allow users to select a value from a gradient. A slider can be placed on the page with a vertical or horizontal orientation.

The slider.htc file can be downloaded from the slider sample page.

The most up to date documentation on the slider behavior can be found on the World Wide Web. Use Internet Explorer to navigate to the behaviors library reference.

If you have the latest MSDN-Online Web Workshop CD (formerly Sitebuilder Web Snapshot CD), map a drive to the CD and use Microsoft Windows Explorer to navigate to \workshop\author\behaviors\library\slider\slider.htm.
