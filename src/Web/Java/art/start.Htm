Microsoft SDK for Java Samples
SDK 2.0 includes a rich set of sample code to be used as reference for the Java application development. Samples fall mainly into two areas: samples for developing applications using the powerful Microsoft class library, especially AFC (Application Foundation Classes), and samples for developing tools that take advantage of the Microsoft Java Virtual Machine run time support.

All sample source code can be found under SDK \Samples directories. Please make sure that you have updated Java VM with the one that was shipped with this release. If you have not done so at the SDK installation time, and need installation instruction, please read the Release Notes . Here are the few requirements for running samples:

  • DirectX3 samples: make sure that you pre-installed the DirectX run time.
  • DirectAnimation samples: available separately by downloading DirectAnimation Samples and Tutorial. DirectAnimation run time is required if you are not IE4 beta user. It is available as seperate donwload file as part of the SDK Active setup.
  • JDBC samples: please use the Intersolv JDBC-ODBC bridge. Please see jdbcrel.htm for installation instructions
  • Native code samples: a C compiler is required.
  • Compiling Samples: make sure you have set your build environment for the languages that you will be using. For example, run the VCVARS32.BAT file that Visual C++ created on your machine, if you will be compiling the samples which include C code.
