Unsigned Applet Installation

The Package Manager can be used to install and run unsigned applets. Unlike IE3, an applet does not need to be signed in order to be installed.

The applet below installs itself using the Package Manager and loads from the local disk each subsequent time this page is reloaded, even if you restart the browser. The applet will update if this web page is updated with a newer version of the applet.

You can see the applet installed as "TreeDemo AFC Sample" in your %WINDIR%\Downloaded Program Files directory.
