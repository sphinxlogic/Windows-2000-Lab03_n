In this example, we have seven SPAN elements, initially set to use the class Item. When the mouse moves over any one of those elements, it will be changed to use the class Highlight. Innovations in Internet Explorer 4 and 5 enable the following:

  • Events can now be generated from a SPAN element. Previously, an HTML author would have had to wrap each SPAN in an anchor to get the right events.
  • With event bubbling, events can be captured at the document level. You do not need to create separate event handlers for each element that will participate in the effect.
