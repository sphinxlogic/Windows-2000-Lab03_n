The ANCHOR Tag

Close This Sample


Data binding is supported to the HREF attribute of an ANCHOR tag. Binding to the ANCHOR tag is supported in both current record and repeated table modes.

An example of a current record bound ANCHOR tag is shown below. Clicking on the arrow buttons will move the current record. This results in the anchor changing to display the values from the current record. Feel free to click on the anchor to navigate to the URL shown. For clarity, a bound SPAN tag is used within the anchor to display the URL.
