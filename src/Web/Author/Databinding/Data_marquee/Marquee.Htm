The MARQUEE Tag

Close This Sample


The contents of the MARQUEE tag can be bound. The marquee can contain either plain text or HTML. Examples of both are shown below, bound to the current record of the data set.

Clicking on the arrow buttons will move the current record. This results in the marquee changing to display the values from the current record.

The MARQUEE tag can also be used in a data bound table. Here's an example:

dataformatas="text" dataformatas="html"
