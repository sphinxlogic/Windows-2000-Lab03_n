This directory contains Platform SDK samples. These samples demonstrate the use of interfaces and APIs for Windows and BackOffice. The samples run on Windows NT and Windows 95. Some of the samples use interfaces or APIs that are only available on certain operating systems or releases. Refer to the readme.txt in the root of the Platform SDK CD for more information about how to build an application targeting the various platforms.

The build environment (headers, libraries, and minimum build tools) for the Platform SDK must be installed in order to build any of the samples beneath this directory. The build environment can be installed by running setup.exe on the root of the Platform SDK CD. The most recent version of the build environment is also available at http://www.microsoft.com/msdn/sdk/platform.htm.

Some components may also require you to install runtimes, depending on the platform on which you are trying to run the sample. Most of these runtimes are found amoung the redistributables are in the \Mssdk\Redist subdirectory on the Platform SDK CD.

Documentation for the components listed above can be installed from the Platform SDK compact disc, or can be viewed online at http://premium.microsoft.com/msdn/library/. This is a free service but registration is necessary.

Read the release notes in ReadMe.Htm and ReadMe.Txt in the root of the Platform SDK compact disc or visit the MSDN web site for more information about the Platform SDK at http://www.microsoft.com/msdn/sdk/
