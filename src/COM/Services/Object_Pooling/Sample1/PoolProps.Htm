Set the "Object Pooling" property

In this step you will set the "Object Pooling" and "Object Construction" properties for:

    • Bank.Account.VC
    • Bank.UpdateReceipt.VC
  1. If not already started, start the Component Services Explorer by selecting Start->Programs->Administrative Tools-> Component Services.
  2. Navigate to and highlight Console Root\Component Services Explorer\Computers\My Computer\COM+ Applications\OPBank\Components.
  3. In the right pane you will see the icon for Bank.Account.VC.
  4. Right click on the icon and select Properties.
  5. Select the Activation tab.
  6. Check the Enable Object Pooling option.
  7. Input the minimum and maximum pool sizes (for example, 1 (min) and 10 (max)). The Creation Timeout can remain at its default.
  8. Check the Enable Object Construction option.
  9. For the Constructor string, type 'FILEDSN=OPBankSamples;DATABASE=pubs;UID=sa;PWD=;APP=OPBank' (without the quotes)
  10. Click OK.
  11. Repeat the steps for the Bank.UpdateReceipt.VC component.

This sequence configures COM+ to create the specified number of pooled components when the component is initially created.
