Create Empty COM+ Applications

In this step you will create 1 empty COM+ application:

    • OPBank
  1. If not already started, start the Component Services Explorer by selecting Start->Programs->Administrative Tools->Component Services.
  2. Navigate to Console Root\Component Services Explorer\Computers\My Computer\COM+ Applications
  3. Right click on the COM+ Applications folder and select New->Application.
  4. When the Wizard appears click Next.
  5. Click the "Create an empty application" button.
  6. When prompted to "Enter a name for the new application", type "OPBank" into the edit box. Make sure that the "Server application" radio button is selected and click Next.
  7. Make sure that the "Interactive user - the current logged on user" radio button is selected and click Next
  8. Click Finish.
