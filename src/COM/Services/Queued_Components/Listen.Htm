Starting a listener for a queued application

  1. If not already started, start the Component Services Explorer by selecting Start->Programs->Administrative Tools->Component Services.
  2. Navigate to and highlight Console Root\Component Services\Computers\My Computer\COM+ Applications
  3. In the right pane select a queued application (such as QCShip).
  4. Right click on the application and select Properties.
  5. Select the Queuing tab.
  6. Select Listen and then click OK.
  7. Right click on the application's icon again and select Start.
