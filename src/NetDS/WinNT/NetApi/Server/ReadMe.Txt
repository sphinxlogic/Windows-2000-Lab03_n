NSXXX Samples

======
SUMMAY
======
The NSXXX samples demonstrate the basic usage
of the networking server set of APIs.


=================
PLATFORM SUPORTED
=================

Windows NT 4 or later.


==================================
HOW TO BUILD AND RUN THE SAMPLE(S)
==================================

To build, type "nmake" at the command line.  For usage of each program, please see the 
corresponding source file.


=====
FILES
=====

The directory contains the following files:

File                       Description

README.TXT                 This file
MAKEFILE                   Project make file
NSDiskEnum.c               NetServerDiskEnum() main program
NSEnum.c                   NetServerEnum() main program
NSGetInfo.c                NetServerGetInfo() main program
NSSetInfo.c                NetServerSetInfo() main program
NSTransEnum.c              NetServerTransportEnum() main program
