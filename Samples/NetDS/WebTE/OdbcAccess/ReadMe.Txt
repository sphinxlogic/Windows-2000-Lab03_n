======================================================
ODBC Access sample database
======================================================

WebTEDB.mdb is an Access sample database that can be used 
as a database for Web Telephony Engine logging. It containes 
a table called WebTETable

To use this database create an entry in the User DSN and point
it to the database. Then, configure WebTE in the administration
tool to use logging to database, and set the DSN name and the table
name as the parameters.
