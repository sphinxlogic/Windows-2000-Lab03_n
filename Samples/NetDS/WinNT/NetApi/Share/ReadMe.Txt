NShXXX Samples

======
SUMMAY
======
The NShXXX samples demonstrate the basic usage
of the networking share set of APIs.


=================
PLATFORM SUPORTED
=================

Windows NT 4 or later.


==================================
HOW TO BUILD AND RUN THE SAMPLE(S)
==================================

To build, type "nmake" at the command line.  For usage of each program, please see the 
corresponding source file.


=====
FILES
=====

The directory contains the following files:

File                      Description

README.TXT                This file
MAKEFILE                  Project make file
NShAdd.c                  NetShareAdd() main program
NShCheck.c                NetShareCheck() main program
NShDel.c                  NetShareDel() main program
NShEnum.c                 NetShareEnum() main program
NShGetInfo.c              NetShareGetInfo() main program
NShSetInfo.c              NetShareSetInfo() main program
