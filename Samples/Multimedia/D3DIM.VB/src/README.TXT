1.  \Mssdk\Samples\Multimedia\D3DIM.VB\src directory
--------------------------------------------------------

The \Mssdk\Samples\Multimedia\D3DIM.VB\src directory contains Visual Basic samples
which use IMControl.Ocx. This file must be generated by building the project located in
\Mssdk\Samples\Multimedia\D3DIM.VB\src\IMControl
