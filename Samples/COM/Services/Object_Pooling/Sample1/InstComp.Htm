For each COM+ application install the appropriate component

In this step we will add the following components:

    • CAccount
    • CGetReceipt
    • CMoveMoney
    • CUpdateReceipt

to the OBBank application.

  1. If not already started, start the Component Services Explorer by selecting Start->Programs->Administrative Tools->Component Services.
  2. Navigate to and highlight Console Root\Component Services Explorer\Computers\My Computer\COM+ Applications\OPBank\Components
  3. Right click on the Components folder and select New->Component.
  4. When the Wizard appears click Next.
  5. Click the "Install new component" button.
  6. Use the dialog to navigate to [Platform SDK Root]\samples\com\services\Object_Pooling\sample1\account.vc\account_ and select Account.dll.
  7. Click Next
  8. Click Finish.
