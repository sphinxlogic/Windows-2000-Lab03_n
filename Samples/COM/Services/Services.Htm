COM+ Services

Overview

COM+ makes it easier to develop, deploy, and administer component-based applications, and lets you leverage current investments in COM and MTS.

New services include:

  • Queuing -- unify objects and messaging
  • Events -- unbound methods
COM+ contains improved services including:
  • Transactions, Servers, Security, Administration
  • Interception
  • Unify COM and MTS for single programming/admin model
  • Interoperability (XA, TIP, TxF, CRM)
  • Backwards compatibility -- 100% compatible with existing COM and MTS components same or better performance

COM+ services offers developers the following benefits:

  • Simplification through reduction
  • Reduce the number of APIs required
  • increase abstraction
  • zero plumbing
  • Simple programming model (clients create, use, release and servers do work)
  • Strong encapsulation

To learn how to use COM+ services, we provide sample applications demonstrating the following features:

  • BYOT
  • Concurrency
  • Compensating Resource Manager
  • Events
  • In-Memory Database
  • Object Pooling
  • Queued Components
  • Resource Dispensers
  • Security
  • Transactions
