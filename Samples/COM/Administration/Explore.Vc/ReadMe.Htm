VCExplore

Overview

The COM+ Explore sample demonstrates how to exercise the COM+ Administration services using Visual C++.

It contains examples of the following:

  • navigating collections (including adding and removing objects and changing properties)
  • exporting and installing applications
  • importing and integrating components into applications
  • controlling the router and IMDB services on the machine
  • writing Start and Stop applications
  • connecting to different machines to administer their COM objects

Build Notes

  • Open the file VCExplore.dsw using Visual C++.
  • Press F7 to build the application or use the Build menu and do a "Build VCExplore.exe".
  • Once the application is built, run the VCExplore.exe application. This can be done by pressing CTRL+F5 or by using the Build menu and select "Execute VCExplore.exe".
  • This application was created using Visual C++ 6.0.
