VBExplore

Overview

The COM+ Explore sample demonstrates how to exercise the COM+ Administration services using Visual Basic.

It contains examples of the following:

  • navigating collections (including adding and removing objects and changing properties)
  • exporting and installing applications
  • importing and integrating components into applications
  • controlling the router and IMDB services on the machine
  • writing Start and Stop applications
  • connecting to different machines to administer their COM objects

Build Notes

  • Open the file Explore.vbp using Visual Basic
  • To build the application, select "Make Explore.exe" from the File menu
  • Select Options... and then select Compile to P-Code
  • This application was created using Visual Basic 6.0
