menu Behavior

This document describes technologies available in Internet Explorer 5. While we encourage you to evaluate these features and to send us your feedback, please note that these features are subject to change.

The menu behavior allows authors to create expanding and collapsing menus on Web pages. Each menu node can enclose other nodes or standard HTML elements, such as an anchor.

The expand.htc file can be downloaded from the menu sample page.

The most up to date documentation on the menu behavior can be found on the World Wide Web. Use Internet Explorer to navigate to the behaviors library reference.

If you have the latest MSDN-Online Web Workshop CD (formerly Sitebuilder Web Snapshot CD), map a drive to the CD and use Microsoft Windows Explorer to navigate to \workshop\author\behaviors\library\menu\menu.htm.
