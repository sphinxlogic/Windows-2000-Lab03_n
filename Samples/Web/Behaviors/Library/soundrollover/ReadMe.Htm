soundRollover Behavior

This document describes technologies available in Internet Explorer 5. While we encourage you to evaluate these features and to send us your feedback, please note that these features are subject to change.

The soundRollover behavior allows Web authors to specify an audio source to be played when the mouse cursor hovers over an element, or when the element is clicked.

The soundRollover.htc file can be downloaded from the soundRollover sample page.

The most up to date documentation on the soundRollover behavior can be found on the World Wide Web. Use Internet Explorer to navigate to the behaviors library reference.

If you have the latest MSDN-Online Web Workshop CD (formerly Sitebuilder Web Snapshot CD), map a drive to the CD and use Microsoft Windows Explorer to navigate to \workshop\author\behaviors\library\soundRollover\soundRollover.htm.
