The SELECT Control

SELECT allows the user to select a single item from a list of pre-defined choices. Authors supply the list of choices in the OPTION tags within the SELECT element. Binding is supported to the VALUE attribute of the selected OPTION. Only single select mode is supported. Both the combo/drop-down and list box formats are supported for display. SELECT can be used with both current record binding and repeated tables.
