The FRAME Tag

Close This Sample
The SRC attribute on the FRAME tag can be bound to a URL in your data set. You can only bind frames to the current record of the data set. The data source object supplying the data set should be present in the HEAD of the HTML page that contains the frameset. Navigation buttons for moving the current record then need to reference this data source object in the parent frameset.

An example of a bound frame is shown on the right. Click on the arrow buttons to change the current record. Notice that the page displayed in the frame on the right changes to match the page at the URL from the data set.
