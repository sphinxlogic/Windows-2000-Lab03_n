The BUTTON Tag

Close This Sample


BUTTON is a new tag introduced in HTML 4.0 that inserts a push button into the page. The button label can be bound to the current record of a data set. The DATAFORMATAS attribute can be used to interpret the bound data as HTML and display it appropriately. If no data is bound to the button, the button label will contain the HTML included between the <BUTTON> and </BUTTON> tags. An example of a button used with current record binding is shown below.

Click on the arrow buttons to change the current record. Notice that the value displayed in the button changes to match the value in the current record.
