You can bind the SRC attribute of an IFRAME to a URL in the column of a data set. The page at the URL specified will be displayed within the IFRAME.

An example of IFRAME used with current record binding is shown below. Click on the arrow buttons to change the current record. Notice that the page displayed in the IFRAME changes to match the value in the current record.
