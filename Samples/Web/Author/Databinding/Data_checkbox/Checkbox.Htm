Check boxes are traditionally used in HTML to determine whether a value is returned to the server on submit. Data binding uses the check box slightly differently. With data binding, the check box binds a boolean value from the data set. This is equivalent to binding the CHECKED attribute rather than the VALUE attribute.

Check boxes can be used with current record binding. An example is shown below. Click on the arrow buttons to change the current record. Notice that the value displayed in the check box changes to match the boolean value of the bound column in the the current record of the data set. A SPAN is also shown bound to the company name in the current record for clarity.
