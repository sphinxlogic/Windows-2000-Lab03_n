Microsoft NetMeeting Web Directory Help

DIRECTORY LIST
User Category - In Microsoft NetMeeting 2.0, each user categorizes his/her NetMeeting usage. You can filter your search to include only those users from a specific category.

Currently in Call - While looking for someone to call, you can filter the search by specifying whether the person you wish to find is already in a call.

Only Show Users with Audio and/or Video Capability - Narrow your search by showing only those with Audio and/or Video capabilities.

Example: You wish to find a co-worker who is already in a call.
Solution: Choose the Business User Category.
Select In a call.
Press the Submit button.
Scroll down in the directory list to look for one of your co-workers.
Click on their name to connect with NetMeeting.

ATTRIBUTES

indicates a user with audio capability.

(Note: NetMeeting 1.0 users and users of NetMeeting compatible products may not show these attributes.)

SEARCH
If you wish to search for a specific person, click on the Search link on the left. There you can search for a user with the specified Name, E-Mail Address, City/State, Country, and Comment. Simply enter search criteria in one or more of the fields.

Example: You wish to find your friend Bob from New York who is logged onto the current ILS server.
Solution: Go to the Search page and type in the name Bob in the First Name box.
Then type in NY in the City/State box.
Also choose United States from the list of countries.
Press the Submit button.
Look for your friend Bob in the resulting list.
Click on his name to connect.

(Note: The City/State and Comment fields are case-sensitive, i.e. the case of the text does matter.)
