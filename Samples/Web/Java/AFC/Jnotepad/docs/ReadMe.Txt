JNotepad Documentation

This directory contains additional documentation on JNotepad. These documents deal with
the technical details of the code and structure of JNotepad. View them if you are interested
in how JNotepad works and how to customize it.

Files in this directory:

JNotepadINI.html			Description of the format JNotepad uses in JNOTEPAD.INI.
					Useful for customizing JNotepad

javadoc\				Directory which contains source code in the JavaDoc
					format. Useful for browsing the class hierarchy of
					JNotepad and for finding out what each class does.
