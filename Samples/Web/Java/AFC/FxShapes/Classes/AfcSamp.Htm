FxShapes Sample Applet

This FxShapes sample shows the various line, shape, and pen styles provided by AFC's graphics and effects classes.

Description

The sample consists of a UIGroup control that contains five panels. The largest panel is the Test Area that displays the options you've selected. By default, the Primitive Type is set to Line, the Pen Style to Solid, and the Pen Width to 4. Click and drag the mouse over the display area to change the position of the line and experiment with different pen styles and widths.

When a different primitive is selected from the list of graphics primitive types, the pen styles and widths for that primitive are enabled (if possible). Those not available are shaded. You can change the position and scale of the primitive by dragging the mouse over the test area. The test area uses an FxTexture as its background, and the Formatted Text option demonstrates both vertical writing and setting text along an outline (FxFormattedText.setOutline).

The Polygon Number of Points option is only enabled when the Polygon primitive type is selected. This option lets you increase or decrease the number of points that form the polygon to a minimum of three or a maximum of ten.
