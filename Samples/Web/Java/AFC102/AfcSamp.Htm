JDK 1.0.2-Compatible AFC Samples

Test drive JDK 1.0.2-compatible AFC by viewing these sample applets.
Please note: these samples illustrate the functionality of JDK 1.0.2-compatible AFC. You must have installed JDK 1.0.2-compatible AFC to run these samples. Please see http://www.microsoft.com/java/afc for more details.

Note: When attempting to run the JDK 1.0.2-compatible AFC samples on Navigator 4.0.3 after they have been installed on your local machine, please use the afcsamp.htm sample page to navigate through the samples. Using FileOpen will result in security or null pointer exceptions being thrown due to a Navigator issue.
