Package Management User Interface

A user can view the current Distribution Units held by the Package Manager using the "Downloaded Program Files" shell extension provided by Internet Explorer 4.0. To use this feature, the user simply opens Explorer and selects the "Downloaded Program Files" subdirectory of their Windows directory.

Each Java Distribution Unit has a small 'J' icon next to it. In the above screenshot, six entries in the list are Java Distribution Units. This view displays the size and version number of each DU.

The user can remove distribution units from their system by right-clicking on an entry and choosing "Remove".

A double-click on a selection will show the Properties window.

Here, the user can view the packages installed and what namespace each package is stored in.

In this case, the above information is for the MiniNotepadApplet, which was installed by the "Signed Applet Installation" page.
