Application Install

The package manager can also be used to install full-blown Java applications. Java applications run by themselves, not inside a Web browser. Internet Explorer can be used to install these applications, however.

When the box below shows the text "Application Installed" it means that the Mapplet application has been installed.

To run Mapplet, open a command prompt and type:

jview -N "Mapplet" Mapplet

which tells JView to run the class Mapplet from application Namespace Mapplet.



Remarks:
Package Management can be used to install not just Java applications, but native Windows applications as well.
