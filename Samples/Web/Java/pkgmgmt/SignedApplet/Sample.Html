Signed Applet Installation

This page installs a signed applet. Once you say "yes" to the security UI which pops up here, the applet and its certificate are stored in the Package Manager. This means that you won't be asked again about the applet until an update appears.

Since this mini-Notepad is signed, you can use it to read files from your local disk. Simply choose "File->Open" from the menu bar.
