    Introduction

    Welcome to the Microsoft Windows 2000 Release Candidate 2 (RC2) Release Edition of the Microsoft Platform Software Development Kit (SDK). This release targets development for the Microsoft® Windows NT® version 4.0, Microsoft Windows® 2000 RC2, and Microsoft Windows 95 and Windows 98 operating systems; the Microsoft BackOffice® version 2.5 family of products; and Microsoft Internet Explorer version 3.x and later.

    The contents of this SDK will change as Windows 2000 evolves. To get the latest released version of the Platform SDK, subscribe to the Microsoft Developer Network (MSDN™) or visit our Web site at http://msdn.microsoft.com/developer/sdk. An MSDN Professional subscription is the best way to receive current Microsoft operating system betas, releases, and the Platform SDK CD-ROM.

    The key objectives of the Platform SDK are to simplify installation of your development environment, introduce you to the latest technologies, and provide information about existing technologies. Visit the MSDN Web site at http://msdn.microsoft.com/developer for the latest information on the Platform SDK and MSDN.

    Installing this SDK
    System Requirements
    • Microsoft® Windows NT® 4.0, Windows® 2000, or Windows 98.
    • Microsoft Visual C++® 5.0 or 6.0 for your x86 or Alpha platform. Additionally, you should install the latest Visual C++ service pack that is available for the version you are using.
    • A CD-ROM drive or access to a network share that contains the contents of the Platform SDK CD-ROM.
    • Approximately 920 MB hard-disk space is required for a full installation. Significantly more space is needed when installing to a FAT partition. However, you may reduce disk space usage by choosing Custom Install and choosing only the components you need.
    • Compiling the entire set of SDK samples requires approximately 1.2 GB of additional disk space.

    SDK Installation
    The Microsoft Platform SDK setup program installs the SDK tools, headers, libraries, and sample files necessary for developing applications for the Windows environment.

    To install the SDK on Windows 98, Windows NT 4.0, or Windows 2000, perform the following steps:

    1. Run setup.exe from the CD-ROM. (Note: if you started readme.htm from your SDK installation, this link to setup.exe will not work. Please start setup.exe from the Platform SDK CD, or use Control Panel, Add/Remove programs to change your current installation.)
    2. Click Next to install the Platform SDK.
    3. Read and accept the End User License Agreement.
    4. Enter your name and organization.
    5. Choose Typical to install the most commonly used options, or Custom to be given a choice of components available for installation.
    6. Choose the installation directory.
    7. If Custom install was chosen, select the components you want to install.
    8. Click Next to begin installation of the Platform SDK files.

    Troubleshooting the SDK Installation
    If the Microsoft Windows Installer update fails and setup cannot continue, please try the following steps:

    1. Restart your computer and then restart the SDK setup program.
    2. If you received a 1603 error from InstMSI.exe and are running Windows 2000 or Windows NT 4.0, try restarting your system and logging on with an account that is a member of the local Administrators group.
    3. Bypass the Microsoft Windows Installer update by double-clicking the appropriate .msi file for your platform (setup\psdk-x86.msi or setup\psdk-Alpha.msi), both of which are found on the Platform SDK CD-ROM. This is not guaranteed to work, and setup behavior will be unpredictable.
